# Future Social 🌍

A simplified social media platform built using the MERN stack with Firebase Auth.

## 🔨 Tech Stack
- React.js
- Node.js + Express
- MongoDB Atlas
- Firebase Auth

## 🚀 Features
- User auth & profile
- Post creation
- Global feed
- Clap-style likes

## 📦 Setup
1. Clone and install
2. Add `.env` in `server/`
3. Run both frontend and backend
